Code 95 is een aanduiding op het rijbewijs. Deze code is in de wereld geroepen om de veiligheid van de Europese transportsector te verbeteren. Niet iedereen heeft een code95 nodig om veilig de weg op te kunnen. In dit artikel gaan we uitleggen wat code95 is en in welke situatie je code 95 nodig hebt.

## **Wat is code95?**

Code 95 is ingevoerd door de Europese Unie om twee belangrijke redenen, namelijk het terugdringen van de CO2-uitstoot en de verkeersveiligheid in Europa verbeteren. Er moest namelijk iets gebeuren aan het aantal verkeersongevallen met vrachtwagens en bussen. Daarom bestaat code95 uit een aantal onderdelen die gericht zijn op verkeersveiligheid.

## **Verplichte onderdelen en nascholing**

Code95 bestaat uit een aantal verplichte onderdelen en een aantal onderdelen die je zelf mag kiezen.

De vrijwillige nascholing die je kunt kiezen als aanvulling op je code95 bestaat uit:

-          Hef en/of reachtruck besturen

-          Het bedienen van de autolaadkraan

-          Werken met een digitale tachograaf (snelheidsmeter)

-          Het zekeren van een lading

-          EHBO

-          Fysieke belasting voorkomen.

Tijdens de verplichte theorie onderdelen leer je alles over de administratie van het goederenvervoer, techniek en verkeer. Het theorie gedeelte sluit je af met een casus administratie VC3 als je vrachtwagenchauffeur bent of VD3 als je buschauffeur bent.

Het praktijkgedeelte van code 95 bestaat uit een besloten terreintoets en de praktische toets C.

## **Wanneer heb je code95 nodig?**

Code95 is verplicht voor iedereen die binnen Europa als beroepsbestuurder wil werken, in de categorie C, C1, D of D1 (evt. aangevuld met E). Om als beroepsbestuurder aan de slag te mogen, moet je kunnen aantonen dat je beschikt over de juiste veiligheidstraining en vakbekwaamheid. Hiervoor heb je dus de code 95 nodig. In dat geval kun je[ nascholing code95 in Maastricht](https://eurotransportcollege.com/nascholing-code-95-maastricht/) nemen.

### **Vrijstelling code 95**

Echter is het in bepaalde situatie mogelijk om vrijstelling te krijgen voor code95. Dit kan bijvoorbeeld als:

-          Er voor privé doeleinden in een vrachtauto of bus wordt gereden

-          Er minder dan 12 uur per week beroepsmatig gereden wordt

-          Het voertuig niet harder dan 45 km p/u kan

-          Je werkzaam bent in een specifieke beroepsgroep, zoals brandweer-, politie- en defensiemedewerkers

-          Je een voertuig bestuurt dat op de weg getest wordt voor reparaties, technische verbeteringen of onderhoud. In dit geval vervoer je namelijk geen goederen of personen

-          Je vervoert een voertuig tijdens een reddingsoperatie, noodtoestand

-          Je vervoert materiaal dat je voor je werk nodig hebt
